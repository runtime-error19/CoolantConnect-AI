# **App Name**: CoolantConnect AI

## Core Features:

- Product Information Display: Showcase radiator coolant products, business information, features, and testimonials from the original marketing page.
- AI Assistant for Queries: Provide an interactive chat experience where users can ask questions about products or services to an AI assistant using the Claude API tool.
- Simulated Support & Group Chat: Allow users to explore pre-configured chat scenarios with a simulated support bot and a group discussion, demonstrating different conversational tones and use cases.
- Dynamic Contact List: Display a sidebar with a list of contacts, showing their online status and latest message, enabling easy switching between chat conversations.
- Search Contacts: Enable users to filter the contact list by name to quickly find a specific chat conversation.
- Conversational UI: Render chat messages in a visually clear bubble format, distinguishing between user and assistant messages, and including a typing indicator.
- Message Input and Sending: Provide an input field and send button for users to compose and dispatch messages within the active chat session.

## Style Guidelines:

- A dark theme predominates, featuring a deep purplish-gray background (#1D1A21) that provides a modern, high-contrast base for content and conversations.
- The primary interactive color is a vibrant indigo-violet (#8E72FF), used for accents, buttons, and user chat bubbles to signify action and personal interaction.
- An energetic cyan (#0DCBF3) serves as an accent, highlighting important elements like active states, key brand colors, and certain informational components, drawing from both original designs.
- Body and headline font: 'Inter' (sans-serif) for its clean, modern, and highly legible characteristics across various text lengths and uses.
- Embrace expressive and easily recognizable emojis for contact avatars and UI action buttons, providing a friendly and intuitive user experience.
- A responsive layout featuring a main content area for the product showcase, with an integrated or toggleable sidebar for chat contacts, allowing for a combined marketing and communication experience.
- Subtle animations like message fading and sliding (fadeSlide) upon appearance and a bouncing effect (bounce) for the 'typing' indicator will enhance the dynamic feel of the chat interface.
